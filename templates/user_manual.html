1. Getting Started

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, or Edge)
  • Internet connection
  • Valid user credentials

Logging In

  1. Navigate to the login page
  2. Enter your username and password
  3. Click the "Login" button

Navigation

The main navigation bar provides access to all major features:

  • Dashboard: Overview of system activity
  • Users: Manage vehicle users and their access
  • Locations: Configure locations and lanes
  • Devices: Manage ANPR cameras and other devices
  • Reports: View and generate various reports

2. Dashboard

Overview

The dashboard provides a real-time overview of system activity, including:

  • Today's access count
  • Access distribution by time
  • Vehicle type distribution
  • Recent access logs

Quick Actions

From the dashboard, you can quickly:

  • View daily reports
  • Check vehicle access status
  • Monitor device status

3. User Management

Adding a New User

  1. Click "Add User" in the Users section
  2. Fill in the required information:
    • Name
    • Vehicle Number
    • Location
    • Valid From and To dates
    • KYC Document
  3. Click "Save" to create the user

Managing User Access

  • Edit user details by clicking the edit icon
  • View user access history
  • Enable/disable user access
  • Update KYC documents

4. Location Management

Adding a Location

  1. Click "Add Location" in the Locations section
  2. Enter location details:
    • Name
    • Address
    • Contact Information
  3. Click "Save" to create the location

Managing Locations

  • Edit location details
  • Add/remove lanes
  • Configure access settings

5. Lane Management

Adding a Lane

  1. Select a location
  2. Click "Add Lane"
  3. Configure lane settings:
    • Lane Name
    • Lane Type (Entry/Exit)
    • Associated Devices
  4. Click "Save" to create the lane

Managing Lanes

  • Edit lane configuration
  • Assign/unassign devices
  • Monitor lane status

6. Device Management

Adding a Device

  1. Click "Add Device" in the Devices section
  2. Enter device details:
    • Device Name
    • Device Type
    • IP Address
    • Port
  3. Click "Save" to add the device

Managing Devices

  • Edit device configuration
  • Assign devices to lanes
  • Monitor device status
  • View device logs

7. Reports

Available Reports

  • Daily Report: Access activity for the current day
  • Weekly Report: Weekly access patterns and statistics
  • Monthly Report: Monthly overview of system activity
  • Vehicle Report: Detailed vehicle access history
  • Location Report: Access patterns by location

Generating Reports

  1. Select the desired report type
  2. Choose the date range (if applicable)
  3. Click "Generate Report"
  4. View or download the report

8. Troubleshooting

Common Issues

  • Login Issues:
    • Ensure correct username and password
    • Check if your account is active
    • Clear browser cache and cookies
  • Device Connection Issues:
    • Verify network connectivity
    • Check device IP and port settings
    • Ensure device is powered on
  • Report Generation Issues:
    • Verify date range selection
    • Check if data exists for the selected period
    • Ensure proper permissions

Getting Help

If you encounter any issues not covered in this manual:

  • Contact your system administrator
  • Check the system logs for error messages
  • Refer to the technical documentation
