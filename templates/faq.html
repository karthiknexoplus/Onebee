Frequently Asked Questions

What is the Access Control System?

The Access Control System is a comprehensive solution for managing vehicle access to facilities. It uses ANPR (Automatic Number Plate Recognition) cameras, Fastag readers, and barrier controllers to automate the entry and exit of vehicles.

How does the system work?

The system works through the following steps:

  1. Vehicle presence is detected by sensors
  2. ANPR camera captures the vehicle number plate
  3. System checks if the vehicle is authorized
  4. If authorized, the barrier opens automatically
  5. Access is logged in the system
What are the main features of the system?

The system includes:

  • Vehicle user management
  • Multiple lane support
  • Device management (ANPR, Fastag, controllers)
  • Access logs and reporting
  • API integration for external systems

How do I add a new vehicle user?

To add a new vehicle user:

  1. Go to the Users page
  2. Click on "Add New User"
  3. Fill in the user details including vehicle number
  4. Upload KYC documents if required
  5. Set validity period
  6. Click "Add User" to save
How do I manage access permissions?

Access permissions can be managed in two ways:

  1. During user creation, select the lanes the user can access
  2. Edit an existing user to modify their access permissions

You can also set specific time restrictions and days of the week for access.

How do I handle expired user accounts?

The system automatically marks users as inactive when their validity period expires. You can:

  • View expired users in the Users list
  • Renew their access by editing their validity period
  • Archive users who no longer need access

How do I add new devices?

To add new devices:

  1. Go to the Settings page
  2. Navigate to the "Devices" tab
  3. Click "Add New Device"
  4. Select the device type (ANPR, Fastag, Controller)
  5. Enter the device details including IP address and port
  6. Assign the device to a lane
  7. Click "Add Device" to save
How do I monitor device health?

The system provides several ways to monitor device health:

  • View device status in the dashboard
  • Check the health status API endpoint
  • Set up alerts for device failures
What should I do if a device is offline?

If a device goes offline:

  1. Check the device's physical connection
  2. Verify the IP address and port settings
  3. Check if the device is powered on
  4. If the issue persists, contact technical support

What types of reports are available?

The system provides several types of reports:

  • Daily Access Report
  • Weekly Access Report
  • Monthly Access Report
  • Vehicle-specific Report
  • Location-specific Report
How do I export reports?

To export reports:

  1. Navigate to the Reports page
  2. Select the report type
  3. Set the date range if applicable
  4. Click "Export" to download the report
How long are access logs retained?

Access logs are retained in the system for 1 year. After that, they are automatically archived. You can export logs before they are archived if you need to keep them for a longer period.

What API endpoints are available?

The system provides several API endpoints:

  • Vehicle presence detection
  • ANPR result processing
  • Barrier control
  • Device health monitoring

You can view the complete API documentation by clicking on "API Docs" in the navigation menu.

How do I integrate with external systems?

To integrate with external systems:

  1. Review the API documentation
  2. Use the appropriate endpoints for your integration
  3. Test the integration in a development environment first
  4. Implement error handling for API responses
Is authentication required for API access?

Yes, API authentication is required for security. You need to include an API key in the request headers. Contact the system administrator to obtain an API key.

What should I do if the system is not recognizing vehicle numbers?

If the system is not recognizing vehicle numbers:

  1. Check if the ANPR camera is functioning properly
  2. Verify that the camera is properly positioned
  3. Ensure there is adequate lighting
  4. Check if the vehicle number is clearly visible
  5. Verify that the vehicle number format is correct in the database
How do I reset the system if it's not responding?

If the system is not responding:

  1. Try refreshing the browser page
  2. Clear browser cache and cookies
  3. If the issue persists, contact technical support
What should I do if I forget my password?

If you forget your password:

  1. Click on "Forgot Password" on the login page
  2. Enter your email address
  3. Follow the instructions sent to your email
  4. If you don't receive the email, contact the system administrator
